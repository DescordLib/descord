# Quickstart
This is an example of how a discord bot made with [descord](https://github.com/thatmagicalcat/descord) should be structured.

## Project structure
```
.
|-- Cargo.toml
|-- README.md
`-- src
    |-- commands
    |   |-- echo.rs
    |   |-- mod.rs
    |   `-- ping.rs
    |-- events
    |   |-- mod.rs
    |   `-- ready.rs
    `-- main.rs
```
